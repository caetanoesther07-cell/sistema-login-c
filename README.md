# Sistema de Login em C

Aplicação desenvolvida em linguagem C que simula um sistema de autenticação de usuário.

## Funcionalidades
- Validação de usuário e senha
- Controle de tentativas
- Bloqueio após erro

## Tecnologias
- Linguagem C

## Como executar

Compile:
gcc login.c -o login

Execute:
./login
